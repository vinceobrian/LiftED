## ğŸ› Troubleshooting

### Page Won't Load
- **Solution**: Use a local server (see options above)
- **Why**: Some features require proper file protocol

### Animations Not Working
- **Check**: Browser supports CSS animations
- **Check**: Reduced motion is not enabled in OS
- **Try**: Refresh the page

### Dark Mode Won't Save
- **Check**: Browser allows localStorage
- **Check**: Not in incognito/private mode
- **Try**: Clear cache and reload

### Mobile Menu Not Appearing
- **Check**: Window width < 768px
- **Try**: Refresh after resizing
- **Look**: Hamburger icon top-right
